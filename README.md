# Advent of Code 2025

Solutions for Advent of Code 2025, written in Python.

## Structure

This repository contains solutions from multiple contributors:

### `daniel/`
Daniel's solutions for each day.

### `mbitzos/`
Mbitzos's solutions for each day.

Each contributor's folder contains day directories (`day01/`, `day02/`, etc.) with:
- `solution.py` - Solution code
- `input.txt` - Puzzle input
- `example.txt` - Example input from the problem description (optional)
