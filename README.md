# Advent of Code 2025

My solutions for Advent of Code 2025, written in Python.

## Structure

Each day has its own directory (`day01/`, `day02/`, etc.) containing:
- `solution.py` - My solution code
- `input.txt` - Puzzle input
- `example.txt` - Example input from the problem description (optional)
